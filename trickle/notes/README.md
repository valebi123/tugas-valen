# Roti Gan - Interactive Website

## Overview
Interactive website untuk Benu Baking Factory yang memperkenalkan brand "Roti Gan" dengan fokus pada tradisi Blitar dan kualitas produk.

## Key Features
- **Dashboard Strategis**: Visualisasi fokus strategis perusahaan menggunakan radar chart
- **Profil Organisasi**: Struktur tim dan informasi lokasi dengan tab interaktif
- **Sistem Pemesanan Online**: Form order dengan integrasi Kurir Qu Kota Blitar dan 3 gerai aktif
- **News Room**: Pusat berita dan aktivitas perusahaan
- **Ulasan Pelanggan**: Form interaktif untuk pelanggan menulis dan membagikan ulasan
- **Responsive Design**: Optimized untuk desktop dan mobile

## Design System
- **Primary Color**: #C19A6B (Tan/Bread)
- **Action Color**: #D2691E (Bronze)
- **Background**: #F9F5F0 (Cream)
- **Typography**: Playfair Display (Headers), Lato (Body)

## Tech Stack
- React 18
- TailwindCSS
- Chart.js for data visualization
- Font Awesome & Lucide Icons

## Structure
- Navigation with mobile menu
- Hero section with CTA buttons
- Dashboard with mission visualization
- Profile with organizational chart
- News room with latest updates
- Footer with contact info
