# Roti Gan - Interactive Company Website

## Overview
Interactive website for Benu Baking Factory's "Roti Gan" brand, showcasing the company's mission, organizational structure, and activities in Blitar, Jawa Timur.

## Key Features
- **Strategic Dashboard**: Visualizes company focus areas using radar chart (Product Quality, Community, HR, Innovation)
- **Organization Profile**: Interactive tabs showing team structure and location information
- **News & Activities**: Latest company news, press releases, and upcoming events
- **Responsive Design**: Mobile-friendly with warm, heritage-inspired color palette

## Technology Stack
- React 18 (production build)
- TailwindCSS for styling
- Chart.js for data visualization
- Lucide icons
